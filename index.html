Skills and Experience

What I Do

Web Development

Transforming Designs into Functional Websites

I specialize in front-end development, crafting interactive and responsive user interfaces using HTML, CSS (Flexbox, Grid), and JavaScript, and building and managing websites on CMS platforms like WordPress, Wix, DevHub, and Drupal, ensuring all websites are validated and meet client specifications and guidelines.

UI/UX Design

Creating User-Centric Digital Experiences

I develop comprehensive UI/UX strategies for brands, produce high-quality design solutions including wireframes, visual designs, flow diagrams, storyboards, site maps, and prototypes, and design and test UI elements such as CTAs, banners, page layouts, and landing pages to enhance user interaction.

Performance Optimization

Ensuring Efficient and Scalable Web Applications

I enhance web application performance and scalability, reduce load times, and improve user experience, while ensuring cross-browser compatibility and adhering to WCAG guidelines to make web applications accessible and functional for users with disabilities.

Tools and Technologies

Utilizing Advanced Tools for Better Web Solutions

I am proficient with Figma and Adobe Photoshop for high-quality design and prototyping, and experienced in jQuery, Bootstrap, ServiceNow Portal, and Google Analytics for comprehensive web development and analysis.
